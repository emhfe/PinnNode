# Pinn Node

## Install

```
npm install
```

## Run

Use two terminals to run web app to see logs, and other for scripts:

### Run web application

```
node server.js
```

Go to `http://localhost:3000/`

### Run scripts

Setting `DEBUG=pinn` will add output to the console.

```
DEBUG=pinn node index.js
DEBUG=pinn node games.js
```
